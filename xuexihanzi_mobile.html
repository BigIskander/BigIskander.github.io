Учить китайские слова.

Это небольшая программа помогающая лучше запоминать (учить) китайские слова. Программа в первую очередь будет полезна людям изучающим китайский язык.
В программе предусмотрены 3 упражнения:

  • Два вида тестов на перевод с русского на китайский и с китайского на русский. В каждом тесте вам из 4 вариантов нужно выбрать 1 верный вариант перевода.
  • И одно упражнение с вопросами открытого типа, где вам нужно самостоятельно с клавиатуры ввести перевод с русского на китайский. Если ни как не получается вспомнить, предусмотрена кнопка показывающая верный перевод (который требуется набрать в поле для ввода текста). Если при этом пользоваться клавиатурой рукописного ввода, это так же может помочь вам повторить (лучше запомнить) написание иероглифов.

Для того чтобы заниматься с программой сначала нужно открыть текстовый файл со словами которые желаете запомнить.
Файл со словам следует разместить во внутренней память телефон, иначе программа не сможет его прочитать.

Правильно оформленный файл со словами, это файл соответсвующий следующим условиям:

  1. это текстовый файл с расширением .txt в кодировке utf-8
  2. в файле не должно быть пустых строк
  3. каждая строка в файле должна содержать два значения: слово на китайском 汉字 и перевод, разделенные знаком табуляции (знак табуляции - это знак, который печатается когда на клавиатуре компьютера нажимаешь клавишу Tab).

Ниже идут примеры файлов со словами для изучения.
Вы можете скачать любой из файлов, нажав правой кнопкой мыши по ссылке и выбрав пункт "Сохранить ссылку как".
Файлы со словами:

  • words1.txt
  • words2.txt
  • words_key.txt
  • 19.txt
  • 20.txt
  • 22.txt
  • 23.txt

После скачивания файл со словам можно, поместить (скоприровать) в память телефона, открыть в программе и начать заниматься (изучать, запоминать слова записанные в файле).

При желании вы можете самостоятельно подготавливать файлы со списками слов для изучения или редакдировать уже существующуюе. Для редактирования файлов со списками слов рекомендуется использовать текстовый редактор Notepad++.

Для большего удобства рекомендуется в настройках программы указать одну из папок в внутренней памяти телефона как папку по умолчанию и загружать все новые файлы со словами туда. Тогда при нажатии кнопки "Выбрать другой файл", сразу будет открываться эта папка по умолчанию и останется только выбрать файл со словами с которым желаете заниматься.

Вы так же можете выгружать слова из сервиса Quizlet.com и создавать из них файлы со словами для изучения.
Для этого:

  • Нужно в Quzlet на странице модуля нажать кнопку с тремя точками и выбрать пункт экспортировать.

  • Далее оставив настройки по умолчанию (как на снимке экрана ниже), нажать кнопку "Копировать текст".

  • Далее в текстовом редакторе создать пустой файл и вставить (ctrl-v) то, что было скопировано (при нажатии кнопки) с сервиса Quizlet. После чего просто сохранить этот файл, с расширением .txt и в кодировке utf-8.

  • Далее этот полученный файл можно так же загрузить в память телефона, открыть в программе и начать заниматься (изучать, запоминать слова записанные в этом файле).

  • Ссылка на учебные модули (наборы слов) на моем аккаунте в Quizlet https://quizlet.com/BigIskander/sets

Надесь для вас эта программа тоже окажется полезной.
